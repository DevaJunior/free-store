# Free-Store | E-commerce Platform

Uma aplicação Front-end moderna desenvolvida em React que simula a experiência completa de um e-commerce, focada em performance e usabilidade.

[Link para o projeto online - Vercel/Netlify]

---

## 🚀 Funcionalidades Principais

* **Catálogo Dinâmico:** Listagem de produtos em grid responsivo com imagens, preços e descrições.
* **Carrinho de Compras:** Sistema completo de adição, remoção e alteração de quantidade de itens, com cálculo em tempo real do subtotal e total.
* **Filtros e Buscas:** Navegação facilitada por categorias de produtos e barra de pesquisa inteligente.
* **Simulação de Checkout:** Fluxo de finalização de compra intuitivo para demonstrar a jornada do usuário (UX).

---

## 💻 Tecnologias e Arquitetura

Este projeto foi construído para demonstrar o gerenciamento de estados complexos e roteamento em uma Single Page Application (SPA):

* **React 18:** Biblioteca base para a construção das interfaces de usuário.
* **Vite:** Ferramenta de build otimizada e ambiente de desenvolvimento ultrarrápido.
* **React Router v6:** Gerenciamento de rotas fluído entre a vitrine, detalhes do produto e o carrinho de compras.
* **Context API / Zustand:** Gerenciamento de estado global para manter as informações do carrinho sincronizadas em toda a aplicação.
* **CSS / Framework de Estilos:** Interface componentizada, mobile-first e acessível.
